# 숫자야구 게임

 ### 숫자야구 게임방식
  #### 게임 승리방식
   - 랜덤하게 정해진 임의의 4자리 숫자를 예측하여 4자리 숫자를 입력하여 결과를 확인한다.
   - 확인한 결과를 토대로 상대의 타깃 숫자를 예상하여 맞힌다.
  #### 게임조건
   - 사용되는 숫자는 0에서 9까지 서로 다른 숫자이다.
   - 입력한 숫자와 타깃숫자를 비교했을때 숫자는 맞지만 위치가 틀렸을때는 볼(BALL)로 나타낸다.
   - 입력한 숫자와 타깃숫자를 비교했을때 숫자와 위치가 전부 일치하면 스트라이크(STRIKE)로 나타낸다.
     - 이때 어떤숫자가 볼이고 스트라이크인지는 알려주지 않고 전체 결과만을 나타낸다.
   - 입력한 숫자와 위치가 전부 틀리면 아웃(OUT)을 나타낸다.
        

 ### 게임 기능
  - 랜덤한 서로다른 네자리의 숫자를 생성하는 기능
  - 화면에서 네자리 숫자를 입력받는 기능
  - 입력받은 숫자를 검사하여 결과를 리턴하는 기능
  - 검사를 통해 받은 결과를 화면에 출력하는 기능
  - 전체 입력한 숫자와 결과를 한번에 확인하는 기능
